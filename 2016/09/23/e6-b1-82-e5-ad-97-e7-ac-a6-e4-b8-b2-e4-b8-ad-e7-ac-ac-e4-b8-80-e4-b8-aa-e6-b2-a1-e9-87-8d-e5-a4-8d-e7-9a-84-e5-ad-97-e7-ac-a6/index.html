求字符串中第一个没重复的字符

这是LeetCode中的题目：387. First Unique Character in a String
给一个字符串，返回字符串中第一个没重复的字符，如果都是重复的则返回－1。
比如：

s = "leetcode"
return 0.

s = "loveleetcode",
return 2.
`</pre>
下面用JavaScript实现：
<pre>`var firstUniqChar = function(s) {
    var s1 = s.split("");
    var s2 = {};
    for(var i=0; i&lt;s1.length; i++){
        var a = s1[i];
        if(!s2[a]){
            s2[a] = 1;
        }else{
            s2[a]++;
        }
    }
    for(var j=0; j&lt;s1.length; j++){
        var b = s1[j];
        if(s2[b] === 1) return j;
    }
    return -1;
}
