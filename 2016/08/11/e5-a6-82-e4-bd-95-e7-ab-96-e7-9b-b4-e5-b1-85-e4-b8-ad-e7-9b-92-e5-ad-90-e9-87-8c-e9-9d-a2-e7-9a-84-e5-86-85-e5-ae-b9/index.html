如何竖直居中盒子里面的内容 

竖直居中一直不是没有水平居中那么容易解决，vertical-align属性总是不那么好掌握。

具体情况如下：一个固定宽高（宽500px，高300px）的div里有一段文字，目标是将那段文字放到div的竖直居中和水平居中的位置。

html代码如下：

<div class=”middle”>
<p>我是竖直居中的文字</p>
</div>

方法一：通过display:table-cell;和virtical-align:middle;完成。

.middle{
display: table-cell;
vertical-align: middle;
text-align: center;
width: 500px;
height: 300px;
background: #ff0;
}

方法二：通过直接设置p元素的line-height属性完成。

.middle{
width: 500px;
height: 300px;
background: #ff0;
text-align: center;
}
.middle p{
line-height: 300px;
}

方法三：通过伪元素 :before 或者 :after 设置。

.middle{
width: 500px;
height: 300px;
background: #ff0;
text-align: center;
font-size: 0;
}

.middle:after{
display: inline-block;
height: 100%;
vertical-align: middle;
content: “”;
}

.middle p{
display: inline-block;
font-size: 14px;
}

方法四：通过display:flex设置。

.middle{
display: flex;
align-items: center;
justify-content: center;
width: 500px;
height: 300px;
background: #ff0;
}

方法五：通过绝对定位设置。

.middle{
position: relative;
width: 500px;
height: 300px;
background: #ff0;
}
.middle p{
position: absolute;
top: 50%;
left: 50%;
display: inline-block;
margin-left: -72px;
margin-top: -8px;
}
