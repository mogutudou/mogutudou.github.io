重新学习之旅

上次说到好多东西都忘了，今天重新开始学习吧，学习的路上我不会停滞不前。

今天把最基础的html翻出来，总结了一个html模板，以后再写也方便多了。

&lt;!DOCTYPE html&gt;
&lt;html lang="zh-cmn-Hans"&gt;
    &lt;head&gt;
      &lt;meta charset="utf-8"&gt;
        &lt;title&gt;网站标题&lt;/title&gt;
      &lt;meta name="description" content="不超过150个字符"&gt;
      &lt;!-- 使用最新的IE和chrome --&gt;
      &lt;meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"&gt;
      &lt;!-- 禁止手机上缩放 --&gt;
      &lt;meta name ="viewport" content ="width=device-width, initial-scale=1, maximum-scale=1, minimum-scale=1, user-scalable=no"&gt;
      &lt;!-- 禁止百度转码 --&gt;
      &lt;meta http-equiv="Cache-Control" content="no-siteapp"&gt;

      &lt;!-- External CSS --&gt;
      &lt;link rel="stylesheet" href="css/normalize.css"&gt;
      &lt;link rel="stylesheet" href="css/main.css"&gt;

      &lt;!-- In-document CSS --&gt;
      &lt;style&gt;
        /* ... */
      &lt;/style&gt;
    &lt;/head&gt;

    &lt;body&gt;

      &lt;!-- JavaScript --&gt;
      &lt;script src="code-guide.js"&gt;&lt;/script&gt;
    &lt;/body&gt;
&lt;/html&gt;

还有html规范确实很重要，我觉得这篇文章写的特别好，有时间的朋友可以看一下编码规范
重新开始学习，每天都进步一点点。
