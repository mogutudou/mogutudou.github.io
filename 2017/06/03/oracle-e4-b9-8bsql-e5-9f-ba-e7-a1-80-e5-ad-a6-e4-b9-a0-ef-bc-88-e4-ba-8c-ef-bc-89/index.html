Oracle之SQL基础学习（二）

接之前Oracle之SQL基础学习（一），之前学习了表空间的一些基本概念以及查看、创建、修改、删除表空间等的一些操作和语句，接下来就开始学习表的相关知识。

表是什么？

表存放在Oracle数据库的表空间里面，是存储数据最基本的存储单位，在Oracle中，表都是二维结构，就是说它是由行和列组成的。

表中的数据类型

字符型：

  • CHAR(n)、NCHAR(n)，这两种是固定长度为n的字符型。
  • VARCHAR2(n)、NVARCHAR2(n)，这两种是可变长度的字符型数据类型，相对节省空间。
    注意：NCHAR(n)、NVARCHAR2(n）这两种是支持Unicode格式进行存放的数据类型。

数值型：

  • NUMBER(p,s)，p可以理解为有效数字，s可以理解为小数点后的位数，也可理解为一个范围。
  • FLOAT(n)，主要用于存储二进制数据。

日期型：

  • DATE(一个范围，精确到秒）
  • TIMESTAMP（精确到小数秒）

其他类型：

  • BLOB
  • CLOB

修改表：
修改表的结构，向表中添加字段，可以使用下面的语句：

<pre>`alter table table_name 
add column_name datatype;
`</pre>

比如，我们可以向刚刚创建好的用户信息表userinfo中添加一个备注字段

<pre>`alter table userinfo
add remarks varchar2(500);
`</pre>

更改字段的数据类型：

<pre>`alter table table_name 
modify column_name datatype;
`</pre>

我们可以更改数据类型的长度，或者直接更换数据类型

删除字段

<pre>`alter table table_name 
drop column column_name;
`</pre>

修改字段的名字（字段类型不变）

<pre>`alter table table_name
rename column column_name to new_column_name;
`</pre>

对表进行改名

<pre>`rename table_name to new_table_name;
`</pre>
